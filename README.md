- Presentation 영상 : [발표영상](https://github.com/jaelyung/HRManagement)

- 프로젝트 개요 : 인사 담당자를 중심으로 한 사원관리 프로그램. 관리자는 사원들의 모든 정보를 등록, 수정, 삭제할 수 있으며 

일반사원들은 민감정보(급여 등)을 제외한 정보조회가 가능함
